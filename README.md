## Getting started
Add the dependency to your `pom.xml` file.
```xml
<repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
</repository>

<dependency>
    <groupId>com.github.dynmie</groupId>
    <artifactId>jeorge</artifactId>
    <version>1.0</version>
</dependency>
```
